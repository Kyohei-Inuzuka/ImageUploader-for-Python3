# ImageUploader-for-Python3
 画像ファイルからサムネイルを作成し、Amazon S3へアップロードを行う Python バッチ

## Description
 Create thumbnails from image files and upload to Amazon S3.

## Requirement
Python 3

## Install
```
cd src
pip install -r requirements.txt
```
